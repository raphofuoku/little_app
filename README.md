## ğŸš€ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/raphofuoku/little_app.git
   cd little_app
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on device/simulator**
   ```bash
   # For iOS
   npm run ios
   
   # For Android
   npm run android
   ```
